How to manage ansible multi versions

  • How to install specific version of ansible
    1. Go to Ansible community changelogs
      1. Get ansible package version ( Ansible Community Package Release)
      1. Get ansible core version (Core version dependency)
    1. Go to Ansible support matrix
      1. Install ansible version supported python (Controller Python)
      1. Get python command path
    curl -s https://gist.githubusercontent.com/suresh-sendinblue/94207cbe3291c19741cb64225b7e7267/raw/47e689875208a64913f3c041ec05dff2c93b9db9/install_ansible_by_version.sh | \ 
bash <Python bin path> <Ansible Core version> <Ansible Package version>

  • How to switch ansible between multiple versions
    1. Add following alias line in $HOME/.bash_rc file
      alias ansibleswitch="source <(curl -s https://gist.githubusercontent.com/suresh-sendinblue/52c00b75bfb6411fde8f864cc7d665de/raw/6f9357e795255cb8693cd8fca58241a5da5bbb63/ansibleswitch.sh)"
    1. Add the source $HOME/.bash_rc line in your $HOME/.profile file at the last
      source $HOME/.bash_rc
    1. Now you can use ansibleswitch command to switch versions
      $ansibleswitch
